# Комментарии
Из стороннего: использован бесплатный шейдер с Asset Store для появления и исчезания куба.
# Описание скриптов
## Environment
Скрипт, инициализирующий игровое пространство. Создает игровое поле и юнитов.
## Tile
Класс тайла
## TileStates
Описание всех состояний тайла
## Unit
Класс юнита
## PathFinding
Класс поиска пути. В данном случае реализован гибрид встроенного NavMesh Agent и волнового алгоритма с эвристикой. NavMesh используется для корректного передвижения юнитов, а волновой алгоритм ищет и строит пути по игровому полю. В качестве эвристики используется дистанция между текущим положением юнита и финальным тайлом.
